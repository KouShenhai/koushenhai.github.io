    4.0.2版本更新日志

    你好呀，我的老朋友！我是老寇，欢迎来到老寇IoT云平台！

    4.0.2版本发布啦！！！

    这个版本包含了许多重要的改进和新功能，旨在提升用户体验和系统性能。

    让我们一起看看有哪些令人兴奋的变化吧！

    # 版本更新

    组件 版本
    Spring Boot 4.0.5
    Spring Framework 7.0.6
    Spring Security 7.0.4
    Spring gRPC 1.0.2
    Spring Data 2025.1.4
    Spring Cloud 2025.1.0
    Spring Cloud Alibaba 2025.1.0.0
    Spring Boot Admin 4.0.2
    Jackson 3.1.2

    Nacos 3.1.0
    gRPC 1.77.0
    Redis 8.0.1

    # 组件更新

    # 协议

    🚀 开源协议兼容Apache 2.0和MIT协议

    # Bug

    🚀 修复一键改包，不修改自动装配的配置文件的路径

    # 后端

    🚀 自定义配置【验证码过期时间，用户信息过期时间】

    🚀 自定义授权码模式登录页面

    🚀 授权码登录支持session-redis

    🚀 国际化菜单增删改查

    🚀 重构Auth模块代码，支持用户密码登录和授权码登录

    🚀 Scalar替换Knife4j

    🚀 增加nacos一键启动的linux脚本

    🚀 重构退出登录【支持密码与授权码注销】

    🚀 使用jackson代替fory，实现redis的序列化与反序列化

    🚀 搭建边缘网关【go实现】【搭建脚手架】

    # 前端

    🚀 API文档

    🚀 国际化菜单管理【查询/保存/修改/删除/查看】

    🚀 用户登录增加默认租户，用户名，密码

    🚀 支持授权码模式登录

    🚀 新增用户需要自行输入密码

    我是老寇，我们下次再见啦！
